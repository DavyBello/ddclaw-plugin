Wrap up this session:

1. Review the entire conversation
2. Identify anything not yet logged:
   - Decisions made
   - Action items created or completed
   - Context shared about people/projects
   - Feedback given or received
   - Anything worth remembering
3. Append to `memory/YYYY-MM-DD.md` (today's date, create if doesn't exist)
4. Use format:
   ```
   ## [Time or topic]
   - bullet points of what happened
   ```
5. Confirm what was saved
